# Referências Bibliográficas

Este documento contém as referências bibliográficas utilizadas no TCC, organizadas por categoria e formatadas segundo as normas ABNT NBR 6023.

---

## Como Citar (Formato ABNT)

### Livros
```
SOBRENOME, Nome. Título: subtítulo. Edição. Local: Editora, Ano.
```

### Artigos de Periódico
```
SOBRENOME, Nome. Título do artigo. Nome do Periódico, Local, v. X, n. X, p. XX-XX, mês ano.
```

### Artigos de Conferência
```
SOBRENOME, Nome. Título do artigo. In: NOME DO EVENTO, número., ano, local. Anais [...]. Local: Editora, ano. p. XX-XX.
```

### Sites e Documentação Online
```
AUTOR. Título da página. Ano. Disponível em: URL. Acesso em: dia mês ano.
```
